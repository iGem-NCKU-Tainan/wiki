PROJECT / Description
U-KNOW - Our Project
Overview

U Know is a portable device combined with urine test strip, providing measurement results to user's smart phone. The test strips are coated with specialized E. coli, which can detect glucose and reponse with RFP accurately. With U Know, we're determined to enhance preventions of diabetes and raise social awareness of this ever-growing issue.

Detection of Glucose

When we started discussing the detection of glucose, we were not sure which kind of indication is more suitable to the device we were trying to make. At the time, we came up with two candidates: amilCP, RFP.
